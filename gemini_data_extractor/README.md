# Gemini Data Extractor

This folder contains the Gemini-based data extraction module and all associated files.

## Files

### Core Module
- `src/gemini_data_extractor.py` - Main extraction module

### Runner Scripts
- `run_extractor_simple.py` - Simple runner script
- `run_gemini_extractor.py` - Full runner script with multiple queries
- `example_gemini_extractor.py` - Example usage


### Documentation
- `GEMINI_DATA_EXTRACTOR.md` - Main documentation
- `GEMINI_*.md` - Additional Gemini-related docs
